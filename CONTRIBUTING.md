# Contributing

Thank you for your interest in contributing to this template!

## Reporting Issues

If you find a bug or have a suggestion for improvement:

1. Check if the issue already exists
2. Create a new issue with a clear description
3. Include steps to reproduce (for bugs)

## Suggesting Enhancements

We welcome suggestions for making this template better:

1. Open an issue describing your enhancement
2. Explain why this enhancement would be useful
3. Provide examples if possible

## Pull Requests

1. Fork the repository
2. Create a new branch for your feature
3. Make your changes
4. Test thoroughly
5. Submit a pull request with a clear description

## Questions?

For questions about React Native Mantine itself, please visit the [main repository](https://github.com/auronsan/react-native-mantine).

Thank you for contributing!
